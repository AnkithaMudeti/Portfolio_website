# Mudeti Ankitha Portfolio

A modern personal portfolio website showcasing Java Full Stack development, professional experience, projects, and contact details.

## Features

- Responsive design
- Dark and Light mode
- Framer Motion animations
- EmailJS contact form
- Resume download
- SEO optimized

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Framer Motion
- EmailJS

## Sections

- Hero
- About
- Skills
- Experience
- Projects
- Achievements
- Education
- Coding Profiles
- Contact

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
